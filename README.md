# 🔓 LINK-BREAKER QR

### El fin del "Tercero". Rompé el bloqueo de links y facturas.

**LINK-BREAKER** es una herramienta de soberanía digital diseñada para permitir el "auto-escaneo". Resolvé el problema de no poder escanear un código QR o de barras que estás viendo en tu propio dispositivo (celular o PC).

## 🚀 ¿Qué hace esta herramienta?

*   **🌐 Escáner QR:** Extraé links de capturas de pantalla de Instagram, Facebook o publicidades digitales donde no podés hacer click.
*   **📦 Escáner de Barras:** Obtené los números de pago de facturas (EDES, Camuzzi, etc.) o códigos de productos directamente desde tu galería de fotos.

## 📸 Cómo se usa (El "Muerde-Codos")
1. Sacá una **captura de pantalla** del código que quieras leer.
2. Entrá a esta App y **subí la imagen**.
3. ¡Listo! Accedé al link o copiá el código de barras al instante.

## 🛠️ Tecnología
Desarrollado con Python, Streamlit, OpenCV y PyZBar. Una solución simple para un problema cotidiano.
